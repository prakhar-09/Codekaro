HOW TO HOST A WEBSITE ON GITHUB

So, before hosting a website one should actually know what is GITHUB. It is a Git repository hosting service, but it adds many of its own features. While Git is a command line tool, GitHub provides a Web-based graphical interface. It also provides access control and several collaboration features, such as a wikis and basic task management tools for every project. It hosting is FREE, SECURE and RELIABLE

STEP 1-You need to create your account on github.com, if you already have an account login else sign up and create one.




STEP 2-Click on create repository, which is on the left hand side.




STEP 3-Then you will be redirected to create a new repository and you have to give a name to your repository .




STEP 4-Then give some description about your repository but it's an optional task.




STEP 5-Then their will be 2 options, one is public and another is private but private is restricted to only Github-Pro members.




STEP 6-After that mark the checkbox and click on the create repository button.




STEP 7-Now click and upload the files.




STEP 8-Now you can just drag and drop your files for uploading them in your repository but this method has a drawback i.e, the max files one can upoad at a time is 100. If you want to upload more then 100 files, use Git commands.




STEP 9-Now your files would start uploading.




STEP 10-After your files are uploaded, click on commit changes.




STEP 11-After that processing starts and this may take a few minute to complete.




STEP 12-After that press on the setting button and scroll down. Their you will find Github Pages which is used to host personal, organizations pages.




STEP 13-Now click on Master Brach and by doing so Github Pages for this repository will be enabled.




STEP 14-After that your site will be ready to get published and you will get its URL.




STEP 15-Reload the page and your webite will be publised.




STEP 16-Copy the url and paste it in a new tab to access your website.




THESE BASIC STEPS WILL HELP YOU TO HOST A WEBSITE ON GITHUB PAGES.
